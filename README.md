# Pyamlenium
A selenium automation tool using YAML format, written in Python.

### Requirements

* Mac OS X 10.13
* Python 3.6^
* Chromedriver


### Setup

Install chromedriver
```
$ brew install chromedriver
```

Install libraries
```
$ # pyenv install 3.6.4
$ pip install -r requirements.txt
```

Launch sample app
```
$ docker-compose up
```

### Usage:

Sample direction
```
$ python -m pyamlenium.runner sample/p1.yaml
```
